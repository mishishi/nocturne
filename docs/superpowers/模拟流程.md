## 第三步：AI追问细节（不收费）

```
收到你的碎片了

帮你追问几个细节
让故事更完整：

1. 梦里是白天还是晚上？
2. 灯是什么颜色的？什么样式？
3. 叫你的人是熟悉的声音吗？
4. 醒来时是什么感觉？（害怕/难过/奇怪）
5. 梦里有没有特别的东西出现？

回答完帮你写成完整故事
这次免费
```

## 第五步：AI生成故事（免费版）

```
你的梦写完了

---

我每次走到那条走廊的第三块地砖就会停下来。

我不知道为什么是第三块。每次脚都会自己停住，像是被什么东西按住了。

灯在闪。老式的白炽灯，电流不稳的时候会这样，唰——一下亮，唰——一下暗下去。我妈的声音又响起来了。

"回来吃饭。"

她的声音从走廊尽头传来，很远，但我听得清清楚楚。每一声都像是从墙壁里面渗出来的。

我想走过去。

但我的脚停住了。就在第三块地砖那里。

低头看了一眼。地上有一双红色的拖鞋，摆得很整齐，鞋尖朝着我来的方向。我从来没见过那双拖鞋。

灯又闪了一下。

这次我看清了。

走廊尽头没有门。墙壁是封死的。就在我妈声音传来的方向，那堵墙是封死的。

但我妈的声音还在响。

"回来吃饭。"

我开始往后退。

不是我想退的。是脚自己开始退的。走廊在变短。我走过的地方在消失。墙壁一块一块地合拢，地砖一块一块地拼回去。

那双红色的拖鞋还在第三块地砖旁边。

两只，整整齐齐。

然后我醒了。

枕头是湿的。窗户外面已经亮了，但我还是觉得冷。好像有什么东西从梦里跟出来了。

我没有去细想那双拖鞋是谁的。

有些东西，不知道比知道好一点。

---

写完了，你看看感觉对不对

这个梦里"追不过去"的感觉很典型
红色的拖鞋在梦里代表一种"无法触及的召唤"
你妈的声音是熟悉的，但那条路是封死的
这种矛盾感可能和你最近的状态有关

你的梦我帮你存档了
以后有新的梦可以继续发给我
```

## 用户决策路径

```
看到朋友圈
↓
免费报名
↓
收到故事
↓
觉得有用 → 自然续费月卡
↓
觉得没用 → 流失

核心：让故事质量说话
```

---

## 这个流程的关键

| 环节 | 重点 |
|------|------|
| 文案 | 不提钱，只说"免费写100个" |
| 追问 | 引导用户思考，不审问式 |
| 故事 | 质量要够好，让用户"哇" |
| 追销 | 轻描淡写，不 push |
| 留存 | 让用户觉得"下次还会来" |

---

## 和之前版本的区别

| 对比项 | 之前 | 现在 |
|--------|------|------|
| 定价时机 | 一开始就提 | 故事满意后再提 |
| 定价话术 | "9.9元/篇" | "49元月卡不限次数" |
| 用户心态 | 花钱买服务 | 先试试看 |
| 转化率 | 低 | 高 |
